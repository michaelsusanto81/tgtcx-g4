# How to use
- Edit **.env** file to match your PostgreSQL DB credentials (refer to **.env.sample**)

- Run go mod vendor
```
go mod vendor
```

- Run backend service
```
make go_run_backend
```

- Run graphql service
```
make go_run_gql
```
